🎁 BLT-Rewards Distribution Service

BLT-Rewards is a blockchain-agnostic rewards distribution service for the OWASP BLT project. This service handles the distribution of rewards tokens across multiple blockchain protocols including Bitcoin Runes (BACON tokens), Solana, and more.

📡 API Endpoints

POST /{protocol}/send-rewards

Send rewards on any supported blockchain protocol.

Supported Protocols: bitcoin-runes, solana

Request Body:

{
  "recipients": [
    {
      "address": "bc1...",
      "amount": 10,
      "token": "BLT•BACON•TOKENS"
    }
  ],
  "fee_rate": 50,
  "dry_run": true,
  "password": "your_password"  // Required when dry_run is false
}
POST /{protocol}/send-rewards-test

Send rewards on testnet/regtest for testing.

Request Body:

{
  "protocol": "bitcoin-runes",
  "num_users": 5,
  "amount_per_user": 1,
  "fee_rate": 50
}
GET /{protocol}/wallet-balance

Get the current wallet balance for a specific protocol.

Example: /bitcoin-runes/wallet-balance

Response:

{
  "success": true,
  "protocol": "bitcoin-runes",
  "balance": "...",
  "tokens": {
    "BLT•BACON•TOKENS": 1000
  }
}
GET /health

Health check endpoint.

Response:

{
  "status": "healthy",
  "service": "BLT-Rewards Distribution Service",
  "version": "1.0.0",
  "supported_protocols": ["bitcoin-runes", "solana"]
}

🚀 Quick Start

Prerequisites

  • Python 3.11+
  • Node.js 14+
  • uv package manager
  • Cloudflare Workers account
  • Protocol-specific blockchain nodes

Installation

# Install dependencies
uv pip install pyyaml

# Configure environment variables
cp .env.example .env
# Edit .env with your configuration

# Deploy to Cloudflare
uv run pywrangler deploy

🔧 Configuration

The service requires the following environment variables:

  • BACKEND_URLS - JSON map of protocol -> backend URL
  • SUPPORTED_PROTOCOLS - Comma-separated list of protocols
  • WALLET_API_PASSWORD - API password for transaction authorization

🔐 Security

Production transactions require a password. Always use "dry_run": true for testing. Never commit credentials to version control.
